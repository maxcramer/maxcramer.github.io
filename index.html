Max Cramer

I have recently been working at Cobidol, as a Junior Full Stack Developer. The entire system was built using AngularJs, which gave me a great opportunity to practice and learn more using Angular. My first project at Cobidol was to build a CACLTV Calculator, which I used Angular Materials for components and styling. After that, I was mainly working on building out reducers, effects, and selectors within the actual project while using Observables and saving data to the Store. I also had an opportunity to work on some tests using Jest Marble. These tests would allow us to create objects and check whether they had the correct properties attached to them.

My interest into web development was recently perked, as previously I had a vegan & vegetarian delivery service with a friend and had yearned for those skills. Having grown up enjoying building websites using online templates, being fascinated by both the programing behind it as well as a love for art and creativity, I decided to grow my skills in web development at General Assembly.

Web development as a career excites me due to the scope of ability to move up within an organisation. I also firmly believe that being able to learn and further those skills while working is a huge bonus that I have not seen in many industries before.

Ideally, I would be working within a close-knit, friendly and patient development team, which would allow me to grow my skills over a long term basis. My dream company would be an ethical one that would help increase people's standard of living or health.

Onewheel Selector

8 Hours | In Free Time

Riding Onewheel has been a passion of mine for a few years now. I have recently noticed though that a lot of people have been looking into getting boards but undecided which, so I built them a form to help them decide. For this, I used Vanilla JS. Each board is initially given a value of 0. When the submit button is clicked, it gives point values to each board depending on the user's selection. When a user changes a single selection and resubmits the form, it reinitializes all the board's values to 0 and reassigns points on the selections to each board. Next, I plan to create an animation that runs for 3 seconds before the results appear. This animation will look as if the program is checking through all the boards to select the correct ones.
